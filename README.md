ScreenToWiki - Screen capture & uploader
===========================================

What's this?
-------------
A tool to capture the screen and upload its image to a wiki.

Environment
------------
* .NET Framework 4.0

How to use?
------------
1. Execute the exe

2. Select the range of screen

3. Input wiki information (URL, username, or password).
